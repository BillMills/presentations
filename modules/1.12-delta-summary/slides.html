Docker 1.12 Delta Enablement

What's New in 1.12?

  • Swarm Mode
  • Swarm Service Abstraction
  • Swarm Security
  • Distributed Application Bundles

Swarm Mode

docker swarm init
  • Dynamically add manager and worker nodes to a swarm
  • Manager redundancy via Raft consensus
  • No more standing up an external key-value store for network state!

Swarm Service Abstraction

docker service create –name frontend –replicas 5 -p 80:80/tcp nginx:latest
docker service scale frontend=100
  • Abstract a group of containers begind a service name
  • Crashed containers automatically replaced
  • Automatic round robin load balancing
  • Scale and update services on the fly

Distributed Application Bundle

  • Definition of networks and images for multi-container app
  • Conceptual heir to Compose
  • Compose generates DAB spec file, but not directly compatible with Swarm Mode
